# Planilhas dos Gráficos de dados coletados do Catálogo da Imprensa da Universidade de Coimbra de 2003 a 2023

Neste diretório, ficam localizadas as planilhas base dos códigos utilizados para a análise de dados do estudo. As planilhas foram organizadas de acordo com o gráfico e o tipo de análise:

**1_Colecoes_e_Series**
    
Apresenta o estudo por série de coleção.

**1_2_Publicacoes_por_Colecao_Serie_Ano**

Apresenta o estudo da quantidade de publicações anuais de acordo com a série.

**1_3_Dispersao_Interativo**

Planilha base para o cálculo de dispersão do número de publicações ao longo dos anos pela IUC.

**2_Publicacoes_Docentes**

Publicações por docente, de acordo com a sua afiliação: universidade, departamento, país, quantidade de publicações dos respectivos catálogos das EUs.
